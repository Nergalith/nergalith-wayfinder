# Demo MBTiles

`demo_jacobabad.mbtiles` is a raster tile package covering a larger **Jacobabad / northern Sindh, Pakistan** test area (regional zoom 7–12 plus local Jacobabad zoom 13 detail) for physical-device field testing.

`demo_bangui.mbtiles` is retained as the original small raster tile package around **Bangui, CAR** (zoom 10–12) for development testing only.

- Generated by `python scripts/generate_demo_mbtiles.py` (OpenStreetMap tiles, attribution in file metadata).
- Jacobabad package generated by `python scripts/generate_jacobabad_mbtiles.py` (OpenStreetMap tiles, attribution in file metadata).
- Original Bangui package is available in the repo as `assets/tiles/demo_bangui.mbtiles`.
- Current APK fallback uses `android/app/src/main/assets/tiles/demo_jacobabad.mbtiles`.
- **Replace before any real deployment.** Production areas use Settings sideload (Phase 2).
