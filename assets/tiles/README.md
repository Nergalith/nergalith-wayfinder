# Demo MBTiles

`demo_bangui.mbtiles` is a small raster tile package around **Bangui, CAR** (zoom 10–12) for development testing only.

- Generated by `python scripts/generate_demo_mbtiles.py` (OpenStreetMap tiles, attribution in file metadata).
- Bundled into the APK via `android/app/src/main/assets/tiles/demo_bangui.mbtiles`.
- **Replace before any real deployment.** Production areas use Settings sideload (Phase 2).
